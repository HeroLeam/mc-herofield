Plugins

Aqui voce irá encontrar todos os plugins instalados no servidor.

ActionHealth é um plugin de exibição de saúde que utiliza a barra de ação para exibir a saúde do jogador e do mob.
Saiba mais.

AutoExp é um plugin que inserirá automaticamente toda a experiência diretamente no seu nível de experiência, em vez de deixar cair um orbe de experiência no chão.
Saiba mais.

AutoPickp é um plugin que coletará todos os itens dropados por você, tanto de blocos, como de mobs.
Saiba mais.

ChestSearch é um plugin para procurar itens nos baús.
Saiba mais.

ChestSort é um plugin para facilitar sua organização em seu inventário e baús.
Saiba mais.

ColoredAnvils é um plugin que permite aos jogadores adicionar cor aos nomes dos itens através do uso de bigornas. É muito simples de usar. A visualização do nome do item é atualizada ao vivo enquanto você digita! Além disso, se você já tiver um item com nome de exibição, poderá repará-lo sem precisar redigitar o nome e ainda manter a formatação!
Saiba mais.

CropChecker é um plugin que permitirá aos jogadores exibir o status de crescimento de qualquer uma de suas colheitas. Simplesmente clicando com o botão direito na cultura com uma enxada, será exibido brevemente na tela o nome e a idade da cultura. A idade é apresentada como uma percentagem de 0% (crescimento não iniciado) a 100% (totalmente crescido). Além disso, você pode clicar com o botão direito em uma Colmeia ou Bee Nest para exibir seu nível de mel, clicar com o botão direito em um Compostor para exibir o quão cheio ele está e até mesmo em um Turtle Egg para exibir seu estado de desenvolvimento.
Saiba mais.

DiscordSRV é um plugin que permite aos jogadores se comunicar tanto pelo jogo como pelo discord, as mensanges são enviadas para as duas interfaces.
Saiba mais.

DKDurability é um plugin que exibirá uma informação quando algo está preste a quebrar.
Saiba mais.

DoorsReloaded é um plugin para melhorar portas vanilla. Você pode abrir/fechar portas duplas com um clique. Isto só funciona se você colocar as portas corretamente (as dobradiças devem estar em lados opostos). Também funciona com Redstone, mas você também pode desativá-lo se usar portas em suas máquinas redstone. Você também pode bater nas portas clicando com o botão esquerdo.
Saiba mais.

Dynmap é um plugin que gera o mapa do servidor de forma online e em tempo real semelhante ao Google Maps que pode ser visualizado AQUI.
Saiba mais.

GlowingItems é um plugin que faz itens como a tocha brilharem quando segurados na mão.
